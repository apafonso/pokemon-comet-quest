# Pokemon Comet Quest

Open [index.html](/Users/anaafonso/Desktop/pokemon-comet-quest/index.html) in a browser to play.

Run in a browser:

```bash
cd /Users/anaafonso/Desktop/pokemon-comet-quest
python3 -m http.server 8002
```

Then visit `http://localhost:8002`.

Run as a desktop app without a browser tab:

```bash
cd /Users/anaafonso/Desktop/pokemon-comet-quest
npm install
npm start
```

That opens the game in its own desktop window using Electron.

Highlights:

- 100 stages across 5 regions, with each region lasting 20 levels.
- 4 playable Pokemon: Pikachu, Bulbasaur, Charmander, and Squirtle.
- Character choice locks for the full campaign until you start a new run.
- Different powers, pickups, and movement styles for each character.
- Commander Keen style progression with comet parts, boss lairs, campaign upgrades, and region-specific adaptation.
- The regions change movement conditions: springs, icy footing, lava vents, storm water zones, and warped gravity.
- Bosses include Meowth, Lapras, Charizard, and a final Mewtwo fight.
